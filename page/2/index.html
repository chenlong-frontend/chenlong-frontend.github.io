浏览器简单数据库模拟(非sql)

发表于 2022-06-15 更新于 2023-08-22 分类于 JavaScript

需求背景

因为 webrtc 项目和 pc 客户端（基于 electron）的，在客户端里，嵌入的是本地网页，里面用到了 electron 底层的数据库存放数据，将这套代码移植到 web 上后，调用底层的 db 方法不存在了，衍生出了这个需求

存储技术选型

因为在后续开发里 db 不再允许使用了，之前的已有由于影响的范围比较广，就没有删除，所以 db 里存储的数据是可控可知的，考虑到数据量比较少，就选用了 localStorage 进行存储

代码实现

主要实现逻辑是，每一个 table 对应一条 localStorage 存储记录的 key，table 里的值用 value 来记录，因为客户端部分调用数据库的时候用的是方法而非直接 sql，所以我这边也无需实现 sql 方法，取而代之的是手动实现 table 的增删改查

这个属于开发中的一个小插曲，感觉比较有意思，记录一下

vue3之flush

发表于 2021-03-10 更新于 2023-08-22 分类于 vue3

注：以下说明基于 vue3 v3.0.7

以下代码，来自于 scheduler

外部调用一般为 queueJob

queueJob 的主要作用就是将传入的任务，压入队列。任务压入完毕之后调用了 queueFlush。

queueFlush 设置了当前等待刷新状态，表明当前有任务正在等待刷新。其后调用了 flushJobs

总结：vue 异步刷新的原理在于，外部的 job 持续进来之后，如果会持续压入队列

如果当前没有任务在等待或者没有正在任务在刷新，就会发起一个微任务，将任务设置为等待状态

如果当前有刷新正在进行，这个job也会顺带着一起处理

由于任务是放在微任务队列里的，所以是异步的
